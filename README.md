# denovo-module-scaffolding
Bioinformatics Training Platform (BTP) Module: Contig Scaffolding

  * Topic
    * Contig Scaffolding
  * Target Audience
    * Biologists
	* Non-bioinformaticians
	* Little to no programming expereience
  * Prerequisites
    * None
